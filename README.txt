Utilize the STAcc_v2 package only after installing the necessary packages in requirements.txt.

The default star tracker is SolarMEMS Technologies' Star Tracker for Small Satellites Miniturization and Robustness, with the pdf located in the main directory.

Remember to first make an account on Space-Track.org in order to access TLEs.

Download the package, navigate to the downloaded folder, and run:
    `streamlit run streamlit.py`
in the terminal.

Please contact kylyn.smith@yale.edu for any inquiries!
